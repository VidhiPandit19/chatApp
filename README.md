# вҡҷпёҸ Installation

Clone the repository:

```bash
git clone https://github.com/VidhiPandit19/chatApp.git
```

Navigate to the project folder:

```bash
cd chatApp
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open the application in your browser:

```
http://localhost:5173
```
